# Branding

## Purpose

`branding/` holds shared brand primitives used across the app, docs, and related surfaces.

## Files

- `links.json`: canonical public URLs for app-to-app navigation
- `tokens.css`: shared high-level color tokens
- `logo/`: reusable logo assets

## Contributor notes

- treat this directory as shared infrastructure for multiple surfaces
- update docs or screenshots when changing user-visible links or branding assets
