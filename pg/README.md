# Fly Postgres Notes

## Purpose

`pg/` documents maintainer access patterns for the separate auth database when it is hosted on Fly Postgres.

## Who this is for

This directory is maintainer-oriented. Outside contributors normally do not need anything here because local auth defaults to sqlite.

## Main use case

This directory exists only as maintainer reference for auth DB access patterns.

## Important note

Do not treat anything in this directory as part of normal public onboarding. It is optional and maintainer-only.
