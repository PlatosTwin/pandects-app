# Support

## Best place to start

For most questions:

- read the root [README.md](README.md)
- check the nearest subsystem `README.md`
- review [ENVIRONMENT.md](ENVIRONMENT.md) for env vars and maintainer-only integrations
- if a directory is git-ignored or obviously local-only, do not treat it as part of the public contributor surface

## Where to ask for help

- Bug reports: open a GitHub issue
- Feature requests: open a GitHub issue
- Security issues: follow [SECURITY.md](SECURITY.md)

## Support model

Pandects is maintained on a best-effort basis. There is no guaranteed SLA for issue triage, debugging help, or feature work.

When asking for help, include:

- what you are trying to do
- which subsystem you are in
- the exact command you ran
- the error output
- whether you are working in public-safe local mode or a maintainer-only workflow
